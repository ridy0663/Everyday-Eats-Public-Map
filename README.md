# Everyday Eats Public Map

Everyday Eats is a food assistance program hosted by Colorado's Department of Human Services (CDHS). This program involves food items being given out at distribution sites and certain times. The program maintains a map of 
active distribution sites to help people get assistance.

Previously, we used a google map for this purpose, which had to be manually maintained by program staff. I was contacted to build an automatically updating replacement. Sadly, department policy prevents me from posting the 
source code to a public repository, but it is built in R Quarto and uses a plotly map with home-brewed accessibility features added.

Check out the map <a href="map.html">here</a>.
